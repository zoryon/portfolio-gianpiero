This is a [Next.js](https://nextjs.org) project starter with [Docker](https://www.docker.com/)

## Getting Started

First, run the development server:

```bash
# automated version
sudo sh start.sh

# manual version
docker compose up
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
